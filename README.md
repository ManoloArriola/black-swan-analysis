# Black Swan Detection in Market Behavior
## Understanding Outliers with Python and Nassim Taleb's Theory

This project is a practical application of **Black Swan Theory**, exploring how high-impact, low-probability events affect market data. While traditional models focus on averages (the "Mediocristan"), this analysis identifies the extreme events that truly drive market shifts (the "Extremistan").

---

## Project Overview
In this notebook, I simulate a retail market environment to:
1. **Identify Outliers:** Using statistical thresholds (Standard Deviation) to find events that defy normal distribution.
2. **Visualize Uncertainty:** Contrasting daily sales averages against unexpected spikes and crashes.
3. **Analyze Impact:** Discussing how "Black Swans" provide more information about market risk than general statistics.

## Tech Stack
* **Language:** Python 3.x
* **Libraries:** * `Pandas`: For data manipulation and DataFrame structures.
    * `NumPy`: For mathematical operations and random data generation.
    * `Matplotlib`: For data visualization and trend plotting.
* **Environment:** Google Colab / Jupyter Notebooks.

## Key Concepts Applied
* **Reproducibility:** Used `np.random.seed` to ensure consistent results for peer review.
* **Statistical Modeling:** Applied the 3-sigma rule to define the boundaries of "normal" behavior.
* **Data Storytelling:** Documenting code with business insights to bridge the gap between programming and management.
